# 📖 KIS-Vibe-Trader 사용자 매뉴얼 (EXE 버전)

본 매뉴얼은 `KIS-Vibe-Trader.exe` 실행 파일을 사용하는 일반 사용자를 위한 가이드입니다. Python 설치 없이 실행 가능하며, TUI(터미널 UI)를 통해 모든 기능을 제어할 수 있습니다.

---

## 1. 시작하기 (Setup)

### A. 준비물
1.  **한국투자증권(KIS) API 키**: [한국투자증권 개발자 센터](https://apiportal.koreainvestment.com/)에서 앱키와 앱시크릿을 발급받으세요. (실전/모의계좌 모두 가능)
2.  **Google Gemini API 키**: [Google AI Studio](https://aistudio.google.com/)에서 발급받으세요. (AI 분석 기능을 위해 필수)

### B. 첫 실행 및 환경 설정
1.  `KIS-Vibe-Trader.exe`를 더블 클릭하여 실행합니다.
2.  최초 실행 시 키보드의 **`S`** 키를 눌러 **[Setup Mode]**로 진입합니다.
3.  화면의 안내에 따라 다음 정보를 입력합니다:
    *   `App Key / Secret`: 한국투자증권에서 발급받은 키.
    *   `Account Number (CANO)`: 계좌번호 8자리.
    *   `Is Virtual? (y/n)`: 모의계좌 사용 여부.
    *   `Gemini API Key`: Google Gemini API 키.
4.  설정이 완료되면 프로그램이 자동으로 재기동되며 실시간 데이터를 수집하기 시작합니다.

## 2. 화면 구성 (Dashboard)

1.  **상단 바**: `[AI TRADING SYSTEM]` 타이틀, 현재 시각 및 한국/미국 시장 개장 여부.
2.  **Market Indices**: KOSPI, KOSDAQ, NASDAQ, 환율 등 주요 지수 실시간 표시.
    *   **C Market**: 비트코인 국내가/해외가 및 **김치 프리미엄(PREM)** 표시.
3.  **VIBE Status**: 현재 시장 분위기 (BULL/BEAR/NEUTRAL/DEFENSIVE) 및 AI 교정 메시지.
4.  **COMMANDS**: 사용 가능한 모든 단축키 리스트.
5.  **전략 정보 패널**:
    *   `STRAT`: 현재 기본 익절/손절 기준 및 Vibe 보정 적용 후 실시간 값.
    *   `BEAR`: 물타기(하락 방어) 전략 설정 — 트리거(%), 회당 금액, 종목한도, 자동 여부. (🔵 청색 계열)
    *   `BULL`: 불타기(수익 확대) 전략 설정 — 트리거(%), 회당 금액, 종목한도, 자동 여부. (🔴 적색 계열)
    *   `ALGO`: AI 추천매매 전략 설정 — 회당 금액, 종목한도, 자동 여부.
    *   각 라인 앞에 `*` 표시가 있으면 초기 설정값에서 변경(AI 또는 수동)되었음을 의미합니다.
6.  **Portfolio List**: 보유 종목의 수익률, 평가금액, 실시간 손익 및 동적 TP/SL(익절/손절선).
7.  **AI Recommendations**: AI가 분석한 테마별 주도주(5개) 및 ETF(1개) 추천 리스트.
8.  **Naver Rankings**: 네이버 금융 기반 실시간 인기/거래량 상위 종목.
    *   **Yesterday's Performance**: 하단에 어제 추천된 종목 중 변동성이 컸던 종목들의 현재 수익률 트래킹 표시.

## 3. 주요 기능 및 단축키

| 키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **`1`** | **수동 매도** | 보유 종목 번호를 선택하여 시장가/지정가로 매도합니다. |
| **`2`** | **수동 매수** | 종목 코드와 수량을 입력하여 즉시 매수합니다. |
| **`3`** | **전략 수정** | 특정 종목의 익절(TP)/손절(SL) 수치를 수동으로 설정합니다. `번호 r` 입력 시 초기화. |
| **`4`** | **AI 추천** | AI 추천 종목의 매매 금액과 자동 모드 여부를 설정합니다. |
| **`5`** | **물타기 설정** | 하락 대응을 위한 물타기 트리거(%), 금액, 한도, 자동 설정. 입력: `트리거% 금액(원) 한도(원) 자동(y/n)` |
| **`6`** | **불타기 설정** | 상승 시 수익 확대를 위한 불타기 트리거(%), 금액, 한도, 자동 설정. 입력 양식은 물타기와 동일. |
| **`7`** | **AI 종목분석** | 종목 코드를 입력하여 뉴스/공시/지표 기반 심층 분석 리포트를 생성합니다. |
| **`8`** | **AI 시황분석** | Gemini AI를 호출하여 현재 시장을 분석하고 익절/손절/물타기/불타기 전략 수치를 자동 도출합니다. |
| **`B`** | **보유 종목 리포트** | 현재 보유 중인 종목에 대한 AI 포트폴리오 진단 리포트를 확인합니다. (유지/매도/비중확대 의견) |
| **`D`** | **추천 종목 리포트** | AI 추천 종목에 대한 상세 분석 리포트를 확인합니다. (PER/PBR/뉴스 통합 분석) |
| **`H`** | **인기 테마 리포트** | 실시간 인기 검색 TOP 10 종목의 테마 분석 및 AI 트렌드 진단 리포트를 확인합니다. |
| **`S`** | **환경 설정** | API 키, 계좌 정보, 전략 파라미터를 다시 설정합니다. |
| **`Q`** | **종료** | 프로그램을 안전하게 종료합니다. |
| **`9`** | **시장 필터** | (숨김 기능) 코스피, 코스닥, 전체 시장으로 랭킹 필터를 변경합니다. |

## 4. 핵심 시스템 가이드

### 자산 현황 (ASSETS) 해석
*   **Total**: (보유 주식 평가액 + 예수금) 합계로, 실제 내 계좌의 총 자산 가치입니다.
*   **Stock**: 현재 보유 중인 모든 주식의 평가 금액 합계입니다.
*   **Cash**: 즉시 매수에 사용 가능한 **예수금(D+2 정산 기준)**입니다.
*   **PnL**: 총 매입 원금 대비 실시간 손익 금액 및 수익률입니다.

### VIBE 기반 동적 리스크 관리
*   **상승장(Bull)**: 시스템이 자동으로 익절선을 상향(+3.0%)하여 수익을 극대화합니다.
*   **하락장(Bear)**: 손절선을 타이트하게 관리(-2.0%)하고 보수적인 익절 전략을 취합니다.
*   **방어모드(Defensive)**: 극보수적 대응으로 모든 임계치를 강하게 조입니다(-3.0%).
*   **패닉(Panic)**: 미국 지수나 비트코인이 급락할 경우 모든 신규 매수를 즉시 차단합니다.

### 물타기 (하락 방어 전략) — `5` 키
*   보유 종목이 하락할 때 평단가를 낮추기 위해 추가 매수하는 전략입니다.
*   트리거 조건이 충족되면 (자동 모드 ON 시) 시스템이 자동으로 집행합니다.
*   **안전장치**: 물타기 트리거는 항상 손절선(SL)보다 높게 유지됩니다.

### 불타기 (수익 확대 전략) — `6` 키
*   보유 종목이 상승 추세일 때 수익 비중을 확대하기 위해 추가 매수하는 전략입니다.
*   상승장(Bull) 또는 거래량 폭발 시에만 작동하며, 직전 매수가 대비 +2.0% 이상 추가 상승했을 때 집행합니다.
*   **안전장치**: 불타기 트리거는 항상 익절선(TP)보다 -1.0% 이상 낮게 제한되어, 매수 직후 익절에 걸리는 핑퐁 매매를 방지합니다.

### AI 자율 매매 (Auto Mode)
*   `4`번 키를 통해 자동 모드를 `ON`으로 설정하면, AI가 발굴한 저평가/모멘텀 종목을 시스템이 스스로 분할 매수합니다.

### AI 시황 분석 (`8`) 전략 자동 적용
*   `8`번 키를 누르면 Gemini AI가 현재 시장을 분석하여 **익절/손절/물타기/불타기/금액** 수치를 한 번에 제안합니다.
*   제안된 수치는 시스템에 즉시 자동 반영되며, VIBE 보정을 역산하여 기본값(Base)으로 저장합니다.
*   AI가 비정형 응답(예: "총자산의 20%")을 보낼 경우, 해당 항목만 기존 설정을 유지하고 나머지는 정상 적용됩니다.

## 5. 문제 해결 (FAQ)

*   **Q: 데이터가 갱신되지 않아요.**
    *   A: 인터넷 연결을 확인하거나, KIS API 키가 만료되지 않았는지 확인하세요. `S` 키를 눌러 설정을 재검토할 수 있습니다.
*   **Q: AI 분석(7/8)이 동작하지 않아요.**
    *   A: `GOOGLE_API_KEY`가 올바르게 입력되었는지 확인하세요. 무료 버전의 경우 초당 호출 제한이 있을 수 있습니다. AI 장애 시 기존 알고리즘 모드로 자동 전환됩니다.
*   **Q: EXE 파일 실행 시 경고 창이 떠요.**
    *   A: 서명되지 않은 실행 파일의 경우 윈도우에서 경고가 뜰 수 있습니다. [추가 정보] 클릭 후 [실행]을 누르세요.
*   **Q: 화면이 깨져 보여요.**
    *   A: 터미널 창 크기를 조절하면 자동으로 다시 그려집니다. 전체 화면 모드(리포트/분석) 후 ESC로 복귀 시에도 자동 갱신됩니다.
