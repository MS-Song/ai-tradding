# 📑 Spec: Group 3 - Advanced Strategy & UI Clarification

## 1. 개요
전략 설정의 편의성을 높이고, 시장 상황(VIBE)과 시간대별 전략(PHASE)의 의미를 명확히 분리하여 사용자에게 전달합니다.

## 2. 주요 기능
### A. 일괄 AI 전략 할당 (요구사항 3)
- **9:전략 확장**: 번호 입력 없이 엔터 입력 시, 현재 보유한 모든 종목을 순회하며 Gemini가 최적 전략을 자동 할당.
- **매수 시 자동화**: 
    - 수동/자동 매수 시 별도 팝업 없이 AI가 즉시 해당 종목에 적합한 프리셋 전략을 선택하여 적용.

### B. VIBE & PHASE 분리 표시 (요구사항 8)
- **표시 양식 변경**:
    - `VIBE`: 현재 시장 지수 기반 분위기 (상승/하락/보합).
    - `PHASE`: 시간대별 관리 정책 (공격/수익확정 등).
- **출력 예시**: `VIBE: NEUTRAL (보합장) [AI: 일치] | PHASE: 🧘CONVERGENCE (횡보장 대응 타이트한 관리)`
- **로직 수정**: PHASE3(수익 확정 시점) 도달 시 VIBE와 관계없이 강제 익절/본전 스탑 로직 발동 확인.

### C. 매뉴얼 정리 (요구사항 7)
- **M:매뉴얼 최적화**: 기술적 동기화 내용은 제거하고, **커맨드 명령어/발동 조건/자동 매매 규칙** 위주로 재구성.

## 3. 전략 관리 정책
- **우선순위**: 종목별 프리셋 전략 > 글로벌 VIBE 보정 > 시간대별 PHASE 보정.
- **PHASE 3 강제성**: 14:30 이후 수익권 종목 50% 분할 매도 로직이 정상 동작하도록 `ExitManager` 검토.
