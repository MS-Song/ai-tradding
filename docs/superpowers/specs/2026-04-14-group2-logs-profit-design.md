# 📑 Spec: Group 2 - Trading Logs & Profit Tracking

## 1. 개요
사용자가 과거 거래 내역을 한눈에 파악할 수 있도록 전용 로그 화면을 신설하고, 실제 체결가 기반의 실적 집계를 구현합니다.

## 2. 주요 기능
### A. L:로그 메뉴 신설 (요구사항 2)
- **전용 화면 (Log View)**: `L` 키 입력 시 리포트 화면과 유사한 별도 창 생성.
- **로그 유형 구분**: 
    - `TRADE`: 수동매입, 수동매도, AI매수, AI매도, 물타기, 불타기
    - `CONFIG`: 설정 변경 (TP/SL 수정 등)
- **표시 규칙**:
    - 거래 로그: 최근 20개, 시간 역순.
    - 설정 로그: 최근 5개, 시간 역순.
    - **실제 체결가 적용**: 시장가 주문 후 API를 통해 반환받은 실제 매수/매도 가격 기록.

### B. 금일 수익금 집계 (요구사항 9-총수익)
- **Asset 패널 확장**: `Asset` 라인 끝에 `금일수익금: +50,000원` 형식으로 표기.
- **계산 로직**: 당일 발생한 `TRADE` 로그 중 매도 체결 데이터를 합산하여 계산.

## 3. 로그 저장 구조 (`trading_logs.json`)
- 별도 파일로 분리하여 관리 (성능 및 관리 용이성).
```json
{
    "trades": [
        { "type": "AI매도", "time": "2026-04-14 10:30:05", "code": "005930", "name": "삼성전자", "price": 82500, "qty": 10, "memo": "TP달성" }
    ],
    "configs": [
        { "time": "2026-04-14 09:00:01", "content": "시스템 시작: Phase 1 진입" }
    ]
}
```
